Homework #3

Guidelines
  • Complete your work on green engineering paper
  • Staple in upper left if more than one sheet.
  • Format the top of the first sheet as follows.
    ECE 383 [Your name] HW#1 [Due Date] [Page ref]
    The page reference for the first page of a four page document would be 1/4, the second page of this document would be labeled 2/4 and so on.
  • Do not write on the back side of the green engineering paper
  • Digital copies of all code, testbenches, and waveforms will also be submitted via BitBucket.


  1. Draw a hardware schematic, similar to the one at the end of lecture 3, for the following circuits. You are given comparators, muxes and adders; do NOT show the internal organization of these devices. Whenever possible reduce the number of devices required to realize the design. You should assume that X, Y, and Z are unsigned(7 downto 0).
    1. if (X==0) then Z = X else Z = Y
    2. if (X==Y) then Z = Y else Z = X+Y
    3. if (X < Y) then Z = X+4 else Z = Y+6
    4. if (X > Y) then Z = X+5 else Z = X+6
  2. Create a digital circuit that takes as input a 8-bit unsigned value (provided by the DIP switches) and illuminates an LED if the input is a multiple of 17. Do NOT use the remainder or division operations, this can easily be accomplished using a single conditional signal assignment statement.
    • A hardcopy (plus digital copy via BitBucket) of your VHDL file (include a proper header).
    • A hardcopy (plus digital copy via BitBucket) of a timing diagram from Isim showing several inputs and the output.
    • A hardcopy (plus digital copy via BitBucket) of your UCF file.
    • Digital copy of your BIT file via BitBucket.
    • Demo your circuit in class on LSN 4.
